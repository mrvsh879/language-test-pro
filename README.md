# Language Test Pro

Профессиональный браузерный тест уровня языка для предварительной оценки кандидатов.

## Что входит

- адаптивный маршрут по уровням CEFR A1–C2;
- отдельная оценка grammar, vocabulary и reading;
- анкета кандидата;
- случайный порядок вопросов и вариантов;
- фиксация времени, уходов со вкладки и подозрительно быстрых ответов;
- диапазон уверенности вместо одного необоснованно точного уровня;
- HR-отчёт и выгрузка результата в JSON;
- печать результата в PDF через браузер;
- полностью статическая версия, совместимая с GitHub Pages.

## Запуск

Откройте `index.html` через локальный HTTP-сервер или включите GitHub Pages для ветки `main`.

Например:

```bash
python -m http.server 8000
```

После этого откройте `http://localhost:8000`.

## Важно

Текущая версия содержит демонстрационный немецкий банк вопросов. Перед использованием при найме банк должен пройти проверку преподавателем или носителем языка. Сам продукт не заявляет официальную сертификацию CEFR.
