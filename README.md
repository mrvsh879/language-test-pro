# Language Test Pro

Профессиональный браузерный инструмент для предварительной языковой оценки кандидатов.

## Готовые языки

- Deutsch
- English
- Polski
- Čeština
- Slovenčina
- Français
- Italiano
- Română

Интерфейс доступен на русском, украинском и английском языках.

## Возможности

- адаптивный маршрут CEFR A1–C2;
- отдельная оценка grammar, vocabulary и reading;
- анкета кандидата и целевой уровень вакансии;
- случайный порядок вопросов и вариантов;
- фиксация времени, уходов со вкладки и слишком быстрых ответов;
- оценочный уровень и диапазон уверенности;
- HR-рекомендация;
- подробный разбор ответов;
- JSON-отчёт и печать/PDF;
- адаптивный дизайн для компьютера и телефона;
- автоматическая публикация через GitHub Pages.

## Запуск локально

```bash
python -m http.server 8000
```

Откройте `http://localhost:8000`.

## GitHub Pages

## Ограничение

Инструмент предназначен для HR-скрининга и не является официальным сертификатом CEFR. Перед высокорисковым использованием банки следует проверить преподавателями и откалибровать на реальных кандидатах.

## Следующие профессиональные модули

- listening с контролем числа прослушиваний;
- writing с оценочной рубрикой;
- speaking с записью ответа;
- backend для одноразовых ссылок и хранения результатов;
- HR-панель и экспорт кандидатов;
- статистическая калибровка сложности заданий.
