# Language Test Pro

Онлайн-сервис для предварительной проверки уровня иностранного языка.

Тест помогает определить примерный уровень кандидата по шкале **CEFR от A1 до C2** и сформировать понятный отчёт для HR или преподавателя.

## Доступные языки

- немецкий;
- английский;
- польский;
- чешский;
- словацкий;
- французский;
- итальянский;
- румынский.

## Что проверяет тест

- грамматику;
- словарный запас;
- чтение и понимание текста;
- понимание речи на слух;
- чтение вслух и произношение.

Сложность заданий постепенно меняется в зависимости от ответов кандидата.

## Результат

После завершения теста формируется отчёт с:

- предполагаемым уровнем CEFR;
- результатами по отдельным навыкам;
- количеством правильных ответов;
- временем прохождения;
- рекомендацией для HR;
- подробным разбором заданий.

## Запуск

Открыть рабочую версию:

**https://mrvsh879.github.io/language-test-pro/**

Для локального запуска:

```bash
python -m http.server 8000
```

После этого открыть:

```text
http://localhost:8000
```

## Основные файлы

```text
index.html          — главная страница
styles.css          — оформление
app.js              — логика теста
data/               — банки вопросов
 audio/             — аудиофайлы для listening
cloudflare/worker.js — проверка speaking
```

## Важно

Тест предназначен для предварительной оценки и HR-скрининга. Он не заменяет официальный языковой экзамен или сертификат CEFR.
